# Progetto_natale2026

Un sito web  per gestire una collezione di carte.

## Funzionalit√†
- Pagina principale: mostra le tue carte e il valore totale della collezione.
- Cerca utenti: trova altri utenti per nome.
- Visualizza collezioni: guarda le carte di altri utenti.

## Come installare ed eseguire


1. Creare un ambiente virtuale nella cartella del progetto:
   ```
   python -m venv .venv
   ```

2. Attivarlo
   - Su Windows: `.venv\Scripts\activate`
   - Su macOS/Linux: `source .venv/bin/activate`

3. Installare le librerie necessarie:
   ```
   pip install -r requirements.txt
   ```
4. avviare il file setupdb.py e poi run.py

5. Apri il browser e vai a http://127.0.0.1:5002/
